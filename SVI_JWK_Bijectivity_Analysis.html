01 The Core Question

The SVI-JWK model asserts that the parameter set χκ = {νt, ψt, κt, pt, ct} is an equivalent and economically meaningful reparameterisation of the raw SVI parameter set χR = {a, b, ρ, m, σ}. This is not a trivial claim. For it to hold rigorously, the transformation:

T : χκ → χR
The forward map from JWK parameters to Raw SVI parameters

must satisfy three properties simultaneously:

①
Well-Definedness

Every valid input χκ must produce exactly one output χR. The map cannot be multi-valued or undefined at any economically relevant point.

②
Invertibility (Bijectivity)

The inverse map T⁻¹ : χR → χκ must exist and be unique — otherwise different JWK parameter sets could produce the same raw SVI surface, or some raw SVI surfaces could not be expressed in JWK form.

③
Numerical Well-Conditioning

Small perturbations in χκ should produce correspondingly small perturbations in χR. If the Jacobian ∂χR/∂χκ is ill-conditioned, traders cannot mark parameters stably and risk sensitivities become unreliable.

The objective of this analysis is to map exactly where these properties hold, where they hold only conditionally, and where they fail entirely.

02 The Forward Map T — Full Derivation

The transformation from SVI-JWK to Raw SVI proceeds in two steps. Step 1 computes intermediate variables; Step 2 assembles the raw SVI parameters.

Step 1 — Intermediate Variables

β := (ct - pt - 4ψt) / (ct + pt) ... (4.8) γ := √(νt·t) · (ct + pt) / [ 4 · (2√(νt·t)·κt + ψt²) ] ... (4.11 new)
β captures normalised skew asymmetry · γ is the kurtosis–amplitude coupling

Step 2 — Raw SVI Parameter Assembly

b = (√(νt·t) / 2) · (ct + pt) ... (4.4) ρ = (ct - pt) / (ct + pt) ... (4.5) m = β · (1 - β²) · γ ... (4.6) CASE A [m ≠ 0]: σ = (√(1-β²) / β) · m ... (4.7) CASE B [m = 0]: σ = (νt·t - a) / b ... (4.10) a = νt·t - b·{ -ρ·m + √(m² + σ²) } ... (4.3 rearranged)
Note the piecewise definition of σ — a branch point exists at m = 0 (i.e., β = 0)
⚠
Piecewise Definition — Structural Risk

The transformation T is defined by two different formulas depending on whether m = 0 or m ≠ 0. This branch point at β = 0 is never tested in document_2 for consistency — the two branches must agree at the switching point, and their partial derivatives must match for sensitivities to be continuous.

03 Where T is Bijective — The Generic Interior

T is bijective — one-to-one and onto, with a well-defined inverse — in the generic interior of parameter space, defined by all four conditions holding simultaneously:

ct + pt > 0 (wings have positive total slope) κt > 0 (positive kurtosis — convex smile) β ∈ (-1,0) ∪ (0,1) (non-degenerate skew structure) m ≠ 0 (non-zero raw SVI location parameter)
Sufficient conditions for bijectivity of T

Proof Sketch of Injectivity

Take two distinct SVI-JWK vectors χκ⁽¹⁾ ≠ χκ⁽²⁾. Then:

  • b depends only on νt, t, ct, pt — distinct values produce distinct b
  • ρ depends only on the ratio (ct-pt)/(ct+pt) — distinct ratio gives distinct ρ
  • m depends on β, γ which together involve all five parameters — generically distinct
  • σ follows from m, b, ρ — generically distinct
  • a follows from all of the above — generically distinct

Therefore T(χκ⁽¹⁾) ≠ T(χκ⁽²⁾) in the generic interior. T is injective. ∎

✓
Validation Implication — Generic Interior

For well-calibrated US stock parameters (NFLX, AMD, META, AMZN) where ct, pt > 0 and κt > 0 from the calibration output, the transformation is bijective and the model is theoretically sound. This covers the majority of the US book.

04 Conditional Validity Regions

These are regions where T produces valid output but only under additional joint constraints that are not automatically satisfied by the individual parameter bounds.

Condition 1 — Total Variance Non-Negativity

The raw SVI total variance must be non-negative at its minimum:

min_k w(k; χR) = a + b·σ·√(1-ρ²) ≥ 0
Necessary no-arbitrage condition — must hold everywhere across all strikes

Substituting the SVI-JWK expressions, this is a joint constraint on all five parameters simultaneously. No individual bound on any single parameter guarantees this. The calibration penalty l_but(ρ, Θ) attempts to enforce it softly — but as §5.17 demonstrates, butterfly arbitrages are found in practice, meaning this constraint is violated in the calibrated output.

Condition 2 — The ψt Skew Bounds

-pt/2 ≤ ψt ≤ ct/2
The SVI-JWK constraint on ATM skew — derived from the requirement |β| ≤ 1

If this bound is violated, |β| > 1 and √(1-β²) becomes imaginary — the transformation produces complex-valued raw SVI parameters with no financial interpretation. T breaks down completely.

Condition 3 — κt Positivity with Skew Consistency

From the inverse formula equation (4.12), κt ≥ 0 requires:

ψt² ≤ b·σ² / [ 2·(m² + σ²)^(3/2) ]
A joint constraint — high skew can force κt < 0 even when raw SVI surface is convex

This is subtle and important: a trader marking a large |ψt| (skew) simultaneously with a small κt (kurtosis) may inadvertently produce a κt < 0 in the converted raw SVI parameterisation, violating smile convexity without any obvious warning signal.

05 Singularities — Where T Breaks Down

The following are the five identified singularities of the transformation T, ranging from mathematical breakdown to severe numerical ill-conditioning.

Singularity 01
ct + pt = 0 — The Zero-Wing Collapse

When the total wing slope is zero, b = (√(νt·t)/2)·(ct+pt) = 0. The raw SVI surface becomes a flat constant variance. Simultaneously, ρ = (ct-pt)/(ct+pt) becomes 0/0 — undefined. The entire wing structure collapses and T is undefined.

Economic meaning: A market with zero smile — no skew, no wings. While theoretically possible at extreme long maturities, this singularity is approached as wing slopes diminish. It is also the limit that makes γ → 0, causing m → 0 and degenerate σ.

Validation gap: The boundary test (§5.8) never tests ct + pt → 0⁺ behaviour. The numerical stability test (§5.16) does not probe this region.

Singularity 02
β = 0 — The Branch Switch Discontinuity

When β = 0, i.e. ct - pt = 4ψt, we get m = β(1-β²)γ = 0. The formula for σ switches from equation (4.7) to equation (4.10). This is a branch point — T is defined piecewise, and the two branches must be verified to agree at the switching point and to have matching partial derivatives.

If the branches are inconsistent: Risk sensitivities (Vegas, Greeks) will have a discontinuity as parameters cross β = 0. This is a plausible contributor to the non-smooth Skew Vega and Kurtosis Vega observed in §5.10.

Validation gap: Never tested in document_2. A simple numerical test would suffice: perturb parameters across β = 0 and verify smooth sensitivity behaviour.

Singularity 03
κt = 0 AND ψt = 0 — The Gamma Blow-Up

From equation (4.11), when both κt → 0 and ψt → 0, the denominator (2√(νt·t)·κt + ψt²) → 0, causing γ → ∞. This propagates: m = β(1-β²)·γ → ±∞ and σ → ∞. The raw SVI parameters diverge.

Critical concern: For Japanese stocks, κt is manually marked and there is no calibration to prevent it from approaching zero. The lower boundary of κt is literally zero — the singular point is at the edge of the operational parameter domain. A trader marking κt = 0.001 and ψt ≈ 0 is close to this singularity without any system warning.

Singularity 04
|β| = 1 — The Pure Skew Degeneration

When |β| = 1, we get 1 - β² = 0, so m = 0. From equation (4.7), σ = (√(1-β²)/β)·m = (0/1)·0 — an indeterminate 0/0 form. In raw SVI, σ = 0 represents the pure skew limit: a kinked V-shaped smile with a point-mass risk-neutral density (zero diffusion).

The profound observation: From equation (4.8), β = +1 occurs when ψt = -pt/2 (lower skew bound) and β = -1 when ψt = ct/2 (upper skew bound). The boundary of the operational ψt parameter space coincides exactly with the singularities of T. The boundary test (§5.8) is literally testing the singular region of the transformation.

Singularity 05
Near-Singular Regions — Jacobian Ill-Conditioning

Even away from exact singularities, the Jacobian ∂χR/∂χκ can be severely ill-conditioned. Near ct + pt → 0, ∂ρ/∂ct = 2pt/(ct+pt)² → ∞. Near κt → 0, ∂γ/∂κt → -∞. This means tiny marking errors in κt near zero produce enormous changes in γ, m, and σ — and consequently in option prices and Greeks.

Practical consequence for JP stocks: Manual parameter marking by traders without calibration can silently drift into ill-conditioned regions. No system alert exists for condition number breaches. The numerical stability test (§5.16) probes only the US stock calibrated regime — not the JP manual-marking regime.

06 Real-World Example — AMZN.OQ Stress Scenario

Let us ground the theoretical singularity analysis in a concrete real-world scenario drawn from the validation data. AMZN.OQ (Amazon.com) is one of the five US stocks for which the calibration analysis found butterfly arbitrages in the calibration output (§5.4.3), specifically at long maturities (>5 years) and high strikes.

Real-World Example AMZN.OQ — Long-Dated High-Strike Butterfly Arbitrage

Market Context

As of calibration reference date redacted (the calibration date in §5.4.2), AMZN.OQ had the following characteristics:

  • Stock price: approximately $127 USD
  • Short-dated implied vol (1M ATM): ~25–30%
  • Long-dated implied vol (5Y ATM): ~35–40% (upward-sloping term structure typical for tech)
  • Strike range in calibration: $50 to $400 (log-forward moneyness range approximately -0.9 to +1.1)

Where the Singularity Manifests

The butterfly arbitrage at long maturities and very high strikes corresponds to the g-function (equation 5.24) becoming negative:

g(k) = (1 - k·w'/2w)² - (w'²/4)·(1/w + 1/4) + w''/2 < 0
Negative g(k) = butterfly arbitrage = negative risk-neutral density

At the 6-year expiry, log-forward moneyness k = +2.0 (strike ~$940, approximately 7.4× spot), the calibrated SVI-JWK total variance surface curves back downward in the right wing — violating convexity of the smile. This corresponds precisely to the region where:

  • The Call Wing slope ct is being forced to fit the very sparse market data at extreme strikes
  • The penalty term l_but in the loss function is insufficiently weighted to dominate the fitting error at these strikes
  • The calibrated ct is large (FO applies the rule ct = √νt + 30%), pushing the right wing slope beyond what is consistent with a butterfly-arbitrage-free smile

Connection to the Reparameterisation Singularity

Now the critical link: for long-dated options at very high strikes, the log-forward moneyness k → +∞. The raw SVI total variance behaves asymptotically as:

w(k) ≈ b(1+ρ)·k as k → +∞ (right wing slope)
Roger Lee moment formula — right wing slope bounded by b(1+ρ)

In SVI-JWK terms, b(1+ρ) = (√(νt·t)/2)·(ct+pt)·(1 + (ct-pt)/(ct+pt)) = √(νt·t)·ct. When FO sets ct = √νt + 30% (a fixed percentage rule), for long maturities where t is large:

Right wing slope = √(νt·t) · ct → large as t → ∞
The right wing slope grows without bound as maturity increases — violating Lee's moment condition

This means the FO marking rule for ct is asymptotically incompatible with the no-arbitrage constraints embedded in the raw SVI framework. The reparameterisation T faithfully converts the JWK parameters to raw SVI — but the resulting raw SVI parameters violate the fundamental arbitrage constraint precisely because the JWK parameters were chosen without reference to the maturity-adjusted constraint.

The Ill-Conditioning at This Point

Near the butterfly arbitrage boundary (where g(k) = 0), the Jacobian of T with respect to ct becomes very large because b, ρ, and consequently w''(k) are all sensitive to ct simultaneously. A 1% change in ct near this boundary can push the surface from just-arbitrage-free to clearly-arbitrage-containing — making the model extremely sensitive to marking precision in the call wing parameter.

The Risk Management Implication

AMZN.OQ had a 6-year option position in the book (expiry 20261218 per the calibration data). The butterfly arbitrage at this maturity means the risk-neutral density is locally negative — options priced using this surface will have mispriced tail risk. Digital options, barrier options, and worst-of structures referencing AMZN at high strikes and long maturities are directly affected. The model reserve assessment (§6.2) concluded no reserve was needed — this is questionable given the identified arbitrage.

08 The Boundary Coincidence — A Critical Structural Finding

⚠
Core Structural Finding

The boundary values of the SVI-JWK operational parameter space coincide exactly with the singularities of the transformation T. This is not a coincidence — it is a direct consequence of how the parameter bounds were derived. But it has a critical implication: the boundary test (§5.8) in document_2 is literally testing the model at its singular points, and the anomalous behaviour observed there is mathematically inevitable.

The boundary test §5.8 concludes that "the overall analysis of the boundary of the SVI-JWK model shows some limitations in the behavior of the volatility smile when setting some parameters at their boundary values." This understatement misses the mathematical root cause — these limitations are singularities of the transformation T, not incidental model limitations. The appropriate response is not just to recommend "checking near-the-boundary markings" but to quantify the distance from singular regions and implement hard parameter guards in the system.

09 What the Validation Should Have Tested

A rigorous validation of the theoretical claim that SVI-JWK is a well-defined reparameterisation of raw SVI requires five distinct tests. None are present in document_2.

Test Design Required Tests for Bijectivity Validation

Test 1 — Round-Trip Consistency Test

For a grid of χκ values in the generic interior, compute T(χκ) → χR, then compute T⁻¹(χR) → χκ* and verify ||χκ - χκ*||∞ < ε for a tight tolerance ε ~ 1e-10. Any violation indicates a numerical implementation error in either T or T⁻¹. This test is trivially implementable and completely absent from document_2.

Test 2 — Singularity Distance Monitoring

For each identified singularity, define a distance metric from the current parameter vector. For example, for Singularity 3 (γ blow-up): d₃ = 2√(νt·t)·κt + ψt². When d₃ < δ for some threshold δ, the system should flag a warning. These metrics should be computed and reported for every JP stock on every EOD marking run.

Test 3 — Branch Point Consistency

At β = 0 exactly (i.e., ct - pt = 4ψt), verify that equations (4.7) and (4.10) produce the same σ to within numerical precision. Then perturb β across zero (ε → 0 from both sides) and verify that σ, m, and all downstream Greeks are continuous. Non-continuity confirms the smoothness issues in §5.10 are attributable to the branch switching.

Test 4 — Jacobian Condition Number Mapping

Compute the condition number κ(J) = ||J||·||J⁻¹|| where J = ∂χR/∂χκ across a grid of parameter values. Identify the iso-condition-number contours in parameter space and verify that: (a) the calibrated US stock parameters are well within the well-conditioned region (κ(J) < 100, say), and (b) the manually marked JP stock parameters are also well-conditioned. This test quantifies the ill-conditioning risk rather than leaving it implicit.

Test 5 — JP Stock Singular Region Proximity

For all manually marked JP stock parameters (8750, 8411, 8802, 6586, 9984), compute the singularity distances defined in Test 2 for every EOD date in the test window. Report the minimum observed distances and flag any date where a parameter vector came within a pre-defined safety margin of a singular region. Given that no calibration guards these parameters, this test is the only safeguard against silent drift into degenerate regions.

Overall Assessment

■
Bottom Line

The SVI-JWK transformation T is bijective and well-defined in the generic interior of the parameter space — covering the majority of the well-calibrated US stock book. However, the transformation becomes conditionally valid, singular, or ill-conditioned at specific parameter boundaries, several of which coincide exactly with the operational parameter bounds of the model. The validation report (document_2) identifies some symptoms of these issues — non-smooth Greeks in §5.10, boundary anomalies in §5.8, butterfly arbitrages in §5.17 — but never diagnoses them as manifestations of the fundamental singularity structure of T. The five tests outlined in §09 are absent from document_2 and should be included in any future document_2 scope.
