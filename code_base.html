1. Purpose and developer workflow

The test fits a QuantLib Andreasen–Huge interpolation to listed option implied-volatility quotes, then exports fitted implied volatility, local volatility, call prices, and basic grid-level static-arbitrage diagnostics. The benchmark is structurally different from SVI-JWK because it is based on an option-price/local-volatility interpolation approach rather than an SVI-family total-variance parameterisation.

Step 1
Install QuantLib-Python and run the sample CSV to confirm environment readiness.
Step 2
Replace sample CSV with production quotes requested from the model owner.
Step 3
Compare AH output to SVI-JWK fitted IVs, prices, densities, and arbitrage diagnostics.

#!/usr/bin/env python3
"""
Andreasen-Huge independent benchmark test for an SVI-JWK volatility surface review.

Purpose
-------
This script fits the QuantLib Andreasen-Huge interpolation to market option implied-volatility
quotes, exports fitted implied volatilities/local volatilities on a test grid, and computes
basic static-arbitrage diagnostics on the benchmark call-price grid.

Regulatory use
--------------
Use this as an independent benchmark diagnostic, not as a replacement production model. The
main validation question is whether SVI-JWK fit/arbitrage findings persist under a structurally
different local-volatility interpolation approach.

Primary implementation references
---------------------------------
1. QuantLib-Python documentation, termstructures/volatility.html:
   ql.AndreasenHugeVolatilityAdapter and ql.AndreasenHugeLocalVolAdapter examples.
2. QuantLib annotated source:
   ql.AndreasenHugeVolatilityInterpl is described as calibration of a local volatility surface
   to a sparse grid of options; it exposes PiecewiseConstant, Linear and CubicSpline modes.
3. Andreasen, J. and Huge, B. (2011), Volatility Interpolation, Risk/SSRN.
4. Le Floc'h / Quants R Us practitioner note: polynomial/cubic interpolation of AH prices may
   reintroduce convexity issues; for validation default to PiecewiseConstant or Linear.

Dependencies
------------
    pip install QuantLib-Python

No pandas dependency is required. CSV input/output uses Python standard library.
"""

from __future__ import annotations

import argparse
import csv
import json
import math
import os
import sys
from dataclasses import dataclass
from datetime import datetime
from typing import Dict, Iterable, List, Optional, Sequence, Tuple

try:
    import QuantLib as ql
except ModuleNotFoundError as exc:
    raise SystemExit(
        "QuantLib-Python is required. Install with: pip install QuantLib-Python\n"
        "This script intentionally uses QuantLib's standard Andreasen-Huge implementation."
    ) from exc


@dataclass(frozen=True)
class QuoteRow:
    as_of_date: str
    underlying: str
    spot: float
    risk_free_rate: float
    dividend_yield: float
    expiry: str
    strike: float
    option_type: str
    implied_vol: float
    quote_source: str

    @property
    def option_enum(self):
        opt = self.option_type.strip().upper()
        if opt in {"C", "CALL"}:
            return ql.Option.Call
        if opt in {"P", "PUT"}:
            return ql.Option.Put
        raise ValueError(f"Unsupported option_type '{self.option_type}'. Use CALL/PUT or C/P.")


def parse_args() -> argparse.Namespace:
    parser = argparse.ArgumentParser(
        description="Fit Andreasen-Huge benchmark and export diagnostics."
    )
    parser.add_argument("--input", required=True, help="CSV quote file.")
    parser.add_argument("--output-dir", default="ah_benchmark_output", help="Output folder.")
    parser.add_argument(
        "--interpolation",
        default="PiecewiseConstant",
        choices=["PiecewiseConstant", "Linear", "CubicSpline"],
        help="Andreasen-Huge interpolation type. Default is PiecewiseConstant for validation conservatism.",
    )
    parser.add_argument(
        "--n-grid-points",
        type=int,
        default=500,
        help="QuantLib AH finite-difference grid points. Default: 500.",
    )
    parser.add_argument(
        "--grid-strikes",
        default="",
        help="Optional comma-separated strikes for output grid. If blank, uses unique input strikes.",
    )
    parser.add_argument(
        "--extrapolate",
        action="store_true",
        help="Allow surface extrapolation when querying output grid.",
    )
    return parser.parse_args()


def read_quotes(path: str) -> List[QuoteRow]:
    required = {
        "as_of_date",
        "underlying",
        "spot",
        "risk_free_rate",
        "dividend_yield",
        "expiry",
        "strike",
        "option_type",
        "implied_vol",
        "quote_source",
    }
    with open(path, newline="", encoding="utf-8-sig") as f:
        reader = csv.DictReader(f)
        missing = required - set(reader.fieldnames or [])
        if missing:
            raise ValueError(f"Input CSV missing columns: {sorted(missing)}")
        rows: List[QuoteRow] = []
        for i, r in enumerate(reader, start=2):
            if not r.get("as_of_date") or not r.get("expiry"):
                raise ValueError(f"Row {i}: as_of_date and expiry are mandatory.")
            row = QuoteRow(
                as_of_date=r["as_of_date"].strip(),
                underlying=r["underlying"].strip(),
                spot=float(r["spot"]),
                risk_free_rate=float(r["risk_free_rate"]),
                dividend_yield=float(r["dividend_yield"]),
                expiry=r["expiry"].strip(),
                strike=float(r["strike"]),
                option_type=r["option_type"].strip(),
                implied_vol=float(r["implied_vol"]),
                quote_source=r["quote_source"].strip(),
            )
            if row.spot <= 0 or row.strike <= 0:
                raise ValueError(f"Row {i}: spot and strike must be positive.")
            if row.implied_vol <= 0:
                raise ValueError(f"Row {i}: implied_vol must be positive decimal, e.g. 0.20 for 20%.")
            rows.append(row)
    if len(rows) < 5:
        raise ValueError("At least five option quotes are recommended for this benchmark.")
    return rows


def validate_common_snapshot(rows: Sequence[QuoteRow]) -> None:
    first = rows[0]
    checks = [
        ("as_of_date", first.as_of_date),
        ("underlying", first.underlying),
        ("spot", first.spot),
        ("risk_free_rate", first.risk_free_rate),
        ("dividend_yield", first.dividend_yield),
    ]
    for attr, expected in checks:
        values = {getattr(r, attr) for r in rows}
        if len(values) != 1:
            raise ValueError(
                f"All rows must share one {attr} for a single benchmark run. Found: {sorted(values)}"
            )
    option_types = {r.option_type.strip().upper() for r in rows}
    if len(option_types) > 1:
        print(
            "WARNING: mixed CALL/PUT input detected. The script will request AH CallPut calibration "
            "if the QuantLib binding exposes it; otherwise calibration may fail.",
            file=sys.stderr,
        )


def parse_ql_date(s: str) -> ql.Date:
    dt = datetime.strptime(s, "%Y-%m-%d").date()
    return ql.Date(dt.day, dt.month, dt.year)


def expiry_to_date(today: ql.Date, calendar, expiry: str) -> ql.Date:
    exp = expiry.strip()
    if "-" in exp:
        return parse_ql_date(exp)
    return calendar.advance(today, ql.Period(exp))


def get_interpolation_type(name: str):
    return getattr(ql.AndreasenHugeVolatilityInterpl, name)


def get_calibration_type(rows: Sequence[QuoteRow]):
    opts = {r.option_enum for r in rows}
    if opts == {ql.Option.Call}:
        return getattr(ql.AndreasenHugeVolatilityInterpl, "Call")
    if opts == {ql.Option.Put}:
        return getattr(ql.AndreasenHugeVolatilityInterpl, "Put")
    return getattr(ql.AndreasenHugeVolatilityInterpl, "CallPut")


def build_andreasen_huge(rows: Sequence[QuoteRow], interpolation_name: str, n_grid_points: int):
    validate_common_snapshot(rows)
    today = parse_ql_date(rows[0].as_of_date)
    ql.Settings.instance().evaluationDate = today
    calendar = ql.NullCalendar()
    day_counter = ql.Actual365Fixed()

    spot_quote = ql.QuoteHandle(ql.SimpleQuote(rows[0].spot))
    rates_ts = ql.YieldTermStructureHandle(
        ql.FlatForward(today, rows[0].risk_free_rate, day_counter)
    )
    dividend_ts = ql.YieldTermStructureHandle(
        ql.FlatForward(today, rows[0].dividend_yield, day_counter)
    )

    calibration_set = ql.CalibrationSet()
    for row in rows:
        payoff = ql.PlainVanillaPayoff(row.option_enum, row.strike)
        expiry_date = expiry_to_date(today, calendar, row.expiry)
        exercise = ql.EuropeanExercise(expiry_date)
        calibration_set.push_back((ql.VanillaOption(payoff, exercise), ql.SimpleQuote(row.implied_vol)))

    interpolation_type = get_interpolation_type(interpolation_name)
    calibration_type = get_calibration_type(rows)
    ah = ql.AndreasenHugeVolatilityInterpl(
        calibration_set,
        spot_quote,
        rates_ts,
        dividend_ts,
        interpolation_type,
        calibration_type,
        n_grid_points,
    )
    black_surface = ql.AndreasenHugeVolatilityAdapter(ah)
    local_surface = ql.AndreasenHugeLocalVolAdapter(ah)
    return ah, black_surface, local_surface, today, calendar, day_counter


def black_vol(surface, expiry_date: ql.Date, strike: float, extrapolate: bool) -> Optional[float]:
    try:
        return float(surface.blackVol(expiry_date, strike, extrapolate))
    except Exception:
        try:
            return float(surface.blackVol(expiry_date, strike))
        except Exception:
            return None


def local_vol(surface, expiry_date: ql.Date, strike: float, extrapolate: bool) -> Optional[float]:
    try:
        return float(surface.localVol(expiry_date, strike, extrapolate))
    except Exception:
        try:
            return float(surface.localVol(expiry_date, strike))
        except Exception:
            return None


def option_price(ah, t: float, strike: float, option_type) -> Optional[float]:
    try:
        return float(ah.optionPrice(float(t), float(strike), option_type))
    except Exception:
        return None


def write_csv(path: str, fieldnames: Sequence[str], rows: Iterable[Dict[str, object]]) -> None:
    with open(path, "w", newline="", encoding="utf-8") as f:
        writer = csv.DictWriter(f, fieldnames=fieldnames)
        writer.writeheader()
        for row in rows:
            writer.writerow(row)


def unique_ordered(values: Iterable[object]) -> List[object]:
    out = []
    seen = set()
    for value in values:
        if value not in seen:
            seen.add(value)
            out.append(value)
    return out


def finite_diff_diagnostics(strikes: List[float], call_prices: List[Optional[float]]) -> Dict[str, object]:
    clean = [(k, c) for k, c in zip(strikes, call_prices) if c is not None and math.isfinite(c)]
    if len(clean) < 5:
        return {
            "status": "INSUFFICIENT_POINTS",
            "min_second_difference": "",
            "negative_convexity_count": "",
            "max_positive_first_difference": "",
        }
    ks = [x[0] for x in clean]
    cs = [x[1] for x in clean]

    first_slopes = []
    for i in range(len(ks) - 1):
        h = ks[i + 1] - ks[i]
        if h > 0:
            first_slopes.append((cs[i + 1] - cs[i]) / h)
    max_positive_first = max([s for s in first_slopes if s > 0.0], default=0.0)

    second_values = []
    for i in range(1, len(ks) - 1):
        h1 = ks[i] - ks[i - 1]
        h2 = ks[i + 1] - ks[i]
        if h1 <= 0 or h2 <= 0:
            continue
        left_slope = (cs[i] - cs[i - 1]) / h1
        right_slope = (cs[i + 1] - cs[i]) / h2
        second = 2.0 * (right_slope - left_slope) / (h1 + h2)
        second_values.append(second)
    min_second = min(second_values) if second_values else None
    neg_count = sum(1 for x in second_values if x < -1e-8)
    status = "PASS_BASIC_GRID_CHECKS" if max_positive_first <= 1e-8 and neg_count == 0 else "REVIEW"
    return {
        "status": status,
        "min_second_difference": "" if min_second is None else f"{min_second:.12g}",
        "negative_convexity_count": neg_count,
        "max_positive_first_difference": f"{max_positive_first:.12g}",
    }


def main() -> int:
    args = parse_args()
    rows = read_quotes(args.input)
    os.makedirs(args.output_dir, exist_ok=True)

    ah, black_surface, local_surface, today, calendar, day_counter = build_andreasen_huge(
        rows, args.interpolation, args.n_grid_points
    )

    expiries = unique_ordered(r.expiry for r in rows)
    if args.grid_strikes.strip():
        grid_strikes = sorted(float(x.strip()) for x in args.grid_strikes.split(",") if x.strip())
    else:
        grid_strikes = sorted({r.strike for r in rows})

    quote_fit_rows = []
    for r in rows:
        exp_date = expiry_to_date(today, calendar, r.expiry)
        fitted_vol = black_vol(black_surface, exp_date, r.strike, True)
        t = day_counter.yearFraction(today, exp_date)
        px = option_price(ah, t, r.strike, r.option_enum)
        quote_fit_rows.append(
            {
                "as_of_date": r.as_of_date,
                "underlying": r.underlying,
                "expiry": r.expiry,
                "strike": r.strike,
                "option_type": r.option_type,
                "market_implied_vol": f"{r.implied_vol:.12g}",
                "ah_fitted_implied_vol": "" if fitted_vol is None else f"{fitted_vol:.12g}",
                "abs_vol_error": "" if fitted_vol is None else f"{abs(fitted_vol - r.implied_vol):.12g}",
                "ah_option_price": "" if px is None else f"{px:.12g}",
                "quote_source": r.quote_source,
            }
        )

    write_csv(
        os.path.join(args.output_dir, "ah_quote_fit.csv"),
        [
            "as_of_date",
            "underlying",
            "expiry",
            "strike",
            "option_type",
            "market_implied_vol",
            "ah_fitted_implied_vol",
            "abs_vol_error",
            "ah_option_price",
            "quote_source",
        ],
        quote_fit_rows,
    )

    surface_rows = []
    diagnostics_rows = []
    for expiry in expiries:
        exp_date = expiry_to_date(today, calendar, expiry)
        t = day_counter.yearFraction(today, exp_date)
        call_prices = []
        for strike in grid_strikes:
            bv = black_vol(black_surface, exp_date, strike, args.extrapolate)
            lv = local_vol(local_surface, exp_date, strike, args.extrapolate)
            cp = option_price(ah, t, strike, ql.Option.Call)
            call_prices.append(cp)
            surface_rows.append(
                {
                    "as_of_date": rows[0].as_of_date,
                    "underlying": rows[0].underlying,
                    "expiry": expiry,
                    "time_to_expiry": f"{t:.12g}",
                    "strike": strike,
                    "ah_black_vol": "" if bv is None else f"{bv:.12g}",
                    "ah_local_vol": "" if lv is None else f"{lv:.12g}",
                    "ah_call_price": "" if cp is None else f"{cp:.12g}",
                }
            )
        diag = finite_diff_diagnostics(grid_strikes, call_prices)
        diagnostics_rows.append(
            {
                "as_of_date": rows[0].as_of_date,
                "underlying": rows[0].underlying,
                "expiry": expiry,
                "time_to_expiry": f"{t:.12g}",
                **diag,
            }
        )

    write_csv(
        os.path.join(args.output_dir, "ah_surface_grid.csv"),
        [
            "as_of_date",
            "underlying",
            "expiry",
            "time_to_expiry",
            "strike",
            "ah_black_vol",
            "ah_local_vol",
            "ah_call_price",
        ],
        surface_rows,
    )
    write_csv(
        os.path.join(args.output_dir, "ah_static_arbitrage_grid_checks.csv"),
        [
            "as_of_date",
            "underlying",
            "expiry",
            "time_to_expiry",
            "status",
            "min_second_difference",
            "negative_convexity_count",
            "max_positive_first_difference",
        ],
        diagnostics_rows,
    )

    calibration_error = None
    try:
        ce = ah.calibrationError()
        calibration_error = [float(x) for x in ce]
    except Exception as exc:
        calibration_error = f"calibrationError() unavailable: {exc}"

    summary = {
        "input_file": os.path.abspath(args.input),
        "output_dir": os.path.abspath(args.output_dir),
        "as_of_date": rows[0].as_of_date,
        "underlying": rows[0].underlying,
        "spot": rows[0].spot,
        "risk_free_rate": rows[0].risk_free_rate,
        "dividend_yield": rows[0].dividend_yield,
        "quote_count": len(rows),
        "expiry_count": len(expiries),
        "strike_count_output_grid": len(grid_strikes),
        "interpolation": args.interpolation,
        "n_grid_points": args.n_grid_points,
        "calibration_error_tuple_if_exposed_by_quantlib": calibration_error,
        "diagnostic_files": [
            "ah_quote_fit.csv",
            "ah_surface_grid.csv",
            "ah_static_arbitrage_grid_checks.csv",
        ],
        "validation_note": (
            "PiecewiseConstant is the default in this script because polynomial/spline interpolation of "
            "Andreasen-Huge prices can reintroduce convexity issues between nodes. Review output against "
            "bid/ask tolerances and production SVI-JWK results; this benchmark is diagnostic, not a replacement model."
        ),
    }
    with open(os.path.join(args.output_dir, "ah_benchmark_summary.json"), "w", encoding="utf-8") as f:
        json.dump(summary, f, indent=2)

    print(json.dumps(summary, indent=2))
    return 0


if __name__ == "__main__":
    raise SystemExit(main())

# Andreasen-Huge Independent Benchmark Test

## Installation

```bash
pip install QuantLib-Python
```

## Run with the sample input

```bash
python andreasen_huge_benchmark_test.py \
  --input sample_ah_option_quotes.csv \
  --output-dir ah_benchmark_output \
  --interpolation PiecewiseConstant
```

## Expected outputs

- `ah_quote_fit.csv`: quote-level fitted AH implied vol and option price.
- `ah_surface_grid.csv`: output grid with AH black vol, local vol, and call price.
- `ah_static_arbitrage_grid_checks.csv`: basic grid-level monotonicity/convexity diagnostics.
- `ah_benchmark_summary.json`: run metadata and calibration summary.

## Input conventions

- `implied_vol` must be decimal, e.g. `0.20` for 20%.
- The sample uses the same compact synthetic quote grid shown in the QuantLib-Python Andreasen-Huge documentation.
- For production validation, replace the sample rows with the Model Owner's market quotes, rates, dividends/borrow and source flags.

## Validation caveat

This benchmark is intended to independently challenge the SVI-JWK surface construction. It is not, by itself, proof that Andreasen-Huge should replace SVI-JWK in production.
