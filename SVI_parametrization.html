Part I — Joint Failure Case Identification

00 Preliminary — Full Parameter Constraint Set

The individual boundary tests in §5.8 vary one parameter at a time. The singularities of T are not always triggered by a single parameter hitting its boundary — they are triggered by combinations of parameters reaching critical joint values. Individual tests miss these entirely.

Before identifying joint failure cases, the complete constraint set for SVI-JWK parameters is stated explicitly:

ν_t ≥ 0                                           (C1)
c_t ≥ 0,  p_t ≥ 0                                 (C2)
κ_t ≥ 0                                           (C3)
-p_t/2  ≤  ψ_t  ≤  c_t/2                         (C4)
a + b·σ·√(1-ρ²)  ≥  0                            (C5)
Operational parameter constraints — all must hold simultaneously

The intermediate variables of the transformation T are:

b  =  (√(ν_t·t) / 2) · (c_t + p_t)               (I1)
ρ  =  (c_t - p_t) / (c_t + p_t)                   (I2)
β  =  (c_t - p_t - 4ψ_t) / (c_t + p_t)            (I3)
γ  =  √(ν_t·t)·(c_t+p_t) / [4·(2√(ν_t·t)·κ_t + ψ_t²)]   (I4)
m  =  β·(1 - β²)·γ                                (I5)
σ  =  (√(1-β²) / β)·m    [Case A: m ≠ 0]          (I6a)
σ  =  (ν_t·t - a) / b    [Case B: m  = 0]          (I6b)
a  =  ν_t·t - b·{-ρ·m + √(m² + σ²)}              (I7)
Note the piecewise definition of σ — a branch point exists at m = 0 (i.e., β = 0)

01 Joint Failure Case 1

01
Joint Failure · Wing Collapse
c_t → 0 AND p_t → 0 Simultaneously
b → 0 · ρ = 0/0 undefined · smile collapses to flat

Mathematical Analysis

From (I1) and (I2):

b  =  (√(ν_t·t) / 2)·(c_t + p_t)  →  0

ρ  =  (c_t - p_t) / (c_t + p_t)  →  0/0  (undefined)
The limit of ρ is path-dependent — it can approach any value in [-1, +1]

The limit of ρ depends entirely on the path by which c_t and p_t approach zero simultaneously:

  • c_t = p_t ρ → 0
  • c_t = 2p_t ρ → 1/3
  • c_t = ε, p_t = ε² ρ → 1
  • p_t = ε, c_t = ε² ρ → -1

The limit is path-dependent. ρ can approach any value in [-1, +1] depending on the relative rate at which c_t and p_t approach zero. The raw SVI parameter ρ — and consequently the entire smile shape — is indeterminate in this joint limit.

What Happens to the Smile

As b → 0, the total variance collapses: w(k) → a = ν_t·t for all strikes. Every option regardless of strike prices at ATM volatility. This implies zero skew and zero wing premium across the entire strike range — economically meaningless.

Jacobian Ill-Conditioning

∂ρ/∂c_t  =  2p_t / (c_t + p_t)²

At c_t = p_t = 0.001:
∂ρ/∂c_t  =  2×0.001 / (0.002)²  =  500
A unit change in c_t produces a 500-unit change in ρ — catastrophic ill-conditioning

At c_t = p_t = 0.001, a marking error of 0.001 in c_t produces a 0.5 change in ρ — moving ρ from 0 to 0.5, completely changing the smile asymmetry. In the operational range c_t + p_t = 0.05, the condition number is still 16 — severe ill-conditioning.

Verdict

Joint Failure Case 1 is the most severe singularity. T is undefined at c_t = p_t = 0 and severely ill-conditioned in the joint neighbourhood c_t + p_t < δ for any small δ. No individual boundary test can detect this.

02 Joint Failure Case 2

02
Joint Failure · Gamma Blow-Up
κ_t → 0 AND ψ_t → 0 Simultaneously
γ → ∞ · m → ±∞ · σ → ±∞ · catastrophic floating-point cancellation

Mathematical Analysis

Define the denominator D = 2√(ν_t·t)·κ_t + ψ_t². As κ_t → 0 and ψ_t → 0 jointly, D → 0 and γ → ∞. The rate depends on the path:

Along ψ_t = 0:  D = 2√(ν_t·t)·κ_t → 0 linearly,    γ ~ 1/κ_t
Along κ_t = 0:  D = ψ_t²           → 0 quadratically, γ ~ 1/ψ_t²

Propagation Through the Chain

m  =  β(1-β²)·γ  →  ±∞
σ  =  (√(1-β²)/β)·m  →  ±∞
√((k-m)² + σ²)  ≈  |σ|  →  ∞
w(k) → ∞ or -∞  (both economically impossible)

The ATM Invariance Exception — Catastrophic Cancellation

ATM invariance w(0) = ν_t·t holds exactly. But the blow-up in m and σ is absorbed by a compensating change in a. This requires floating-point arithmetic to subtract two very large numbers of similar magnitude — catastrophic cancellation:

a  =  ν_t·t  -  b·{-ρ·m + √(m² + σ²)}

If m, σ ~ 10⁶:  the term b·{-ρ·m + √(m²+σ²)} involves
cancellation of 10⁶-scale quantities to produce a 10⁰-scale result.

Standard double precision (15-16 sig. digits) loses all accuracy
for m, σ > 10⁸.
ATM vol appears correct while all other strikes are computed incorrectly — silently masking the failure

Condition for Practical Occurrence

For ν_t = 0.09, t = 1.0:

D < 0.01  ⟺  0.6·κ_t + ψ_t² < 0.01

At ψ_t = 0:    κ_t < 0.017
At κ_t = 0:    |ψ_t| < 0.10

Both individually small — well within the operational parameter range
for manually marked JP stocks.
Verdict

A genuine numerical singularity causing catastrophic floating-point cancellation. ATM vol appears correct while all off-ATM strikes are computed incorrectly — the failure is silent. Particularly dangerous for JP stocks where κ_t is manually marked and may approach zero without system alert.

04 Joint Failure Case 4

04
Dropped — Market Infeasible
ν_t → 0 AND ψ_t → 0 (Near-Zero Vol, Near-Zero Skew)
σ indeterminate 0/0 · theoretically possible but no real market analogue

ν_t = σ²_ATM approaching zero means ATM implied volatility approaching zero. In any functioning option market, implied volatility cannot be zero — it would create an immediate arbitrage (buy the option for zero, collect positive payoff at expiry with positive probability).

The lowest ATM implied vols observed in liquid equity markets are approximately 5-8% annualised (e.g. a broad volatility index near 8-9 in a historical low-volatility period, individual stock vols occasionally hitting 10-12%). Dropped

Not feasible as a market scenario. The condition ν_t → 0 has no realistic market analogue in any functioning equity option market. This case is a mathematical boundary condition only.

m  =  ε·(1-ε²)·G  ≈  εG

As ε → 0 and G → ∞ simultaneously:

  εG → 0:     Case B applies,  σ = (ν_t·t - a)/b
  εG → ∞:     Case A applies,  σ = √(1-ε²)·G → G

Three completely different behaviours — the model does not
know which formula for σ to use.
The branch switch itself becomes indeterminate when both singularities are approached simultaneously

F1 Case 1 Market Feasibility — Wing Collapse

Is This Feasible?

Yes — but only in specific structural conditions.

Market Condition: Post-Crisis Vol Compression in Japanese Equities

Japanese single stock implied volatility surfaces are structurally flatter than US equivalents. This is well-documented — Japan has lower equity risk premium, lower stock-specific vol relative to index vol, and a market dominated by institutional investors with limited option buying activity in individual names.

Real Example Japanese financial-sector stock A / Japanese financial-sector stock B — a low-rate low-volatility period low-rate Environment

During a low-rate low-volatility period, Japanese equity realised volatility was at multi-decade lows. The the local equity index 30-day realised vol touched approximately 8-10% annualised. The the local central bank's equity ETF purchases (initiated an earlier policy period, expanded a subsequent policy period and a later policy period) further suppressed individual stock volatility and smile steepness by reducing tail risk perception.

For individual financial sector stocks like Japanese financial-sector stock (Japanese financial-sector stock A), option markets showed near-flat smiles with wing slopes of the order 0.02-0.05 — well within the ill-conditioned region. A trader marking p_t = 0.03 and c_t = 0.04 produces c_t + p_t = 0.07, giving condition number ∂ρ/∂c_t ≈ 12 — severe operational ill-conditioning.

Plausible

Specifically for JP financial sector stocks in low vol environments. Directly applicable to Japanese financial-sector stock A (Japanese financial-sector stock) and Japanese financial-sector stock B (Japanese financial-sector stock B) in the source-document population during low-rate-era flat smile conditions.

F2 Case 2 Market Feasibility — Gamma Blow-Up

Is This Feasible?

Yes — and it is arguably the most practically relevant case.

Market Condition: Long-Dated Options on Low-Vol, Low-Skew Stocks

Near-zero kurtosis and near-zero skew simultaneously describes a smile that is nearly linear — no curvature and no rotation relative to ATM. The term structure of skew is well-known to decay approximately as 1/√T — meaning at 6 years, skew is roughly 0.41× its 1-year value.

Direct Book Reference Japanese technology stock B — 6-Year Position in document_2 Book

Japanese technology stock B is explicitly identified as problematic in document_2. For a 6-year option with 1-year skew ψ_t ≈ -0.05, the 6-year skew would be approximately -0.05/√6 ≈ -0.020 — very close to zero.

A trader marking κ_t = 0.005 and ψ_t = -0.010 for a 5-year Japanese technology stock B option — not unreasonable given the market — produces:

D  =  2√(0.09×5) × 0.005 + 0.010²
   =  2×0.671×0.005 + 0.0001
   =  0.0067 + 0.0001
   =  0.0068  →  γ > 100  (blow-up region)

This is directly applicable to the largest notional position in the source-document population.

Supporting Evidence US Stocks — a historical low-volatility period Low Vol Environment

The a broad volatility index spent extended periods below 12 during a historical low-volatility period. US growth stock C and US growth stock D — both in the document_2 sample — showed reduced skew and kurtosis at medium maturities. US growth stock C 2-year implied skew in a historical low-volatility period was approximately 2-3 vol points per 10% strike — implying ψ_t of the order -0.02 to -0.03. Combined with low manually marked κ_t values, the blow-up condition is reachable.

Plausible — Directly Relevant

Specifically for long-dated options in low vol/low skew environments. Directly applicable to the Japanese technology stock B 6-year position in the source-document population, where the blow-up condition is numerically confirmed.

F3 Case 3 Market Feasibility — Double Singularity

Is This Feasible?

Partially — requires careful qualification.

Market Condition: Stress Event with Manual Marking Lag

During equity market stress events, skew increases sharply while kurtosis may not increase proportionally. Market participants buy OTM puts for downside protection — skew rises — but the buying is concentrated near specific strike levels rather than across all strikes, so kurtosis (curvature at ATM) may not rise proportionally.

Market Reference a broad market stress episode market stress period — Skew Spike with Stale Kurtosis Marks

During the market stress crash, single stock implied vol skew spiked dramatically. For US tech stocks that had been in low-vol environments (low κ_t baseline), the sudden skew spike could push ψ_t toward the -p_t/2 boundary while κ_t remained small because the curvature of the smile had not yet adjusted.

For JP stocks where κ_t is manually marked and may not be updated during fast market moves, a trader who keeps κ_t at its last marked value (small, from the low-vol environment) while updating ψ_t to reflect new market skew could inadvertently approach this joint boundary. The manual marking process creates a lag between skew updates and kurtosis updates.

Plausible with Qualification

Feasible in specific stress scenarios with manual marking lag. Relevant but less common than Cases 1 and 2. The pure joint case of maximum skew combined with zero kurtosis is more constrained — retained with explicit qualification that it requires the combination of a stress event and manual marking lag.
