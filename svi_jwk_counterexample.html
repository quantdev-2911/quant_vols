SVI-JWK Linear Interpolation Counterexample Dataset

Purpose. This report provides synthetic node data where quoted maturity nodes are clean for butterfly and calendar checks, but linear interpolation of SVI-JW/SVI-JWK parameters creates off-node calendar arbitrage.

Convention: raw SVI is defined on total variance w(k). SVI-JW/SVI-JWK values in the CSV are total-variance diagnostics derived from the raw SVI parameters. If a production implementation uses a different JWK convention, the mapping functions must be replaced with the exact production formulas.

1. Raw SVI and SVI-JW/SVI-JWK convention

The raw SVI total variance slice is:

w(k) = a + b [ rho (k - m) + sqrt((k - m)^2 + sigma^2) ]

The report uses the following SVI-JW/SVI-JWK-style diagnostics:

v = w(0) psi = b/(2 sqrt(v)) [ rho - m / sqrt(m^2 + sigma^2) ] p = b(1-rho)/sqrt(v) c = b(1+rho)/sqrt(v) vtilde = a + b sigma sqrt(1-rho^2)

This makes the raw and JWK versions equivalent at the quoted nodes. The failure is introduced only when the JWK vector is linearly interpolated and converted back to raw SVI.

2. Exact no-calendar-arbitrage condition being tested

For each fixed log-moneyness k, total variance should be non-decreasing in maturity:

w(k, T2) - w(k, T1) >= 0, for all k and T2 > T1

The node-level check verifies this only at quoted nodes. The off-node check generates interpolated JWK slices and verifies the same condition along the interpolation path.

7. Interpretation for model validation

The dataset demonstrates that node-level absence of butterfly and calendar arbitrage is not sufficient to validate linear interpolation of SVI-JW/SVI-JWK parameters. The interpolation path itself must be tested.

The relevant production control is:

For each adjacent node pair: 1. interpolate SVI-JW/SVI-JWK parameters across a dense lambda grid; 2. convert each interpolated vector to raw SVI using the production mapping; 3. test w(k, lambda[j+1]) - w(k, lambda[j]) >= -tolerance across strikes; 4. test fixed-strike Black price monotonicity under production forward/dividend conventions; 5. test off-node butterfly diagnostics g(k) >= -tolerance.

The m-shift ratio is also reported. It is an escalation diagnostic, not a published no-arbitrage theorem.

8. Caveats

  • This is a synthetic validation dataset, not a reconstruction of a particular market date.
  • The Black price test assumes F=100 and zero discounting to isolate the interpolation effect. Production testing must use actual forward, discounting, dividend and moneyness conventions.
  • The grid check is numerical. For a stronger review, increase strike density, lambda density, and add local optimisation around the worst grid point.
  • If your production SVI-JWK convention differs, replace the mapping functions before relying on the result.

Appendix B. Reproducibility script: svi_jwk_counterexample_test.py

The full Python script is embedded below. It regenerates the checks, produces the off-node calendar-arbitrage diagnostics, and can be run independently.

#!/usr/bin/env python3
"""
SVI-JWK linear-interpolation counterexample generator and validator.

Purpose
-------
This script demonstrates a deliberately engineered synthetic volatility dataset where:
1. Quoted maturity nodes pass node-level butterfly and calendar-arbitrage checks.
2. Linear interpolation of SVI-JW/SVI-JWK parameters between two adjacent nodes creates
   off-node calendar arbitrage.

Important convention
--------------------
This file uses the standard total-variance raw SVI slice

    w(k) = a + b [ rho (k - m) + sqrt((k - m)^2 + sigma^2) ]

and the SVI-JW/SVI-JWK-style diagnostics

    v       = w(0)
    psi     = b/(2*sqrt(v)) * [rho - m/sqrt(m^2 + sigma^2)]
    p       = b*(1-rho)/sqrt(v)
    c       = b*(1+rho)/sqrt(v)
    vtilde  = a + b*sigma*sqrt(1-rho^2)

The inverse mapping used here is included explicitly. If your production implementation
uses a different SVI-JWK convention, replace raw_to_jw() and jw_to_raw() with the exact
production mapping before using this as a validation test.

Dependencies
------------
numpy, pandas, matplotlib
"""

from __future__ import annotations

import base64
import io
import math
from pathlib import Path
from typing import Dict, Iterable, List, Tuple

import matplotlib.pyplot as plt
import numpy as np
import pandas as pd


OUTPUT_DIR = Path(__file__).resolve().parent
NODE_CSV = OUTPUT_DIR / "svi_jwk_counterexample_nodes.csv"
REPORT_HTML = OUTPUT_DIR / "svi_jwk_counterexample_report.html"

K_GRID = np.linspace(-2.0, 2.0, 1601)
LAMBDA_GRID = np.linspace(0.0, 1.0, 101)
FWD = 100.0
STRIKE_GRID = FWD * np.exp(K_GRID)


def raw_w(k: np.ndarray | float, p: Iterable[float]) -> np.ndarray:
    """Raw SVI total variance w(k)."""
    a, b, rho, m, sigma = [float(x) for x in p]
    k = np.asarray(k, dtype=float)
    x = k - m
    return a + b * (rho * x + np.sqrt(x * x + sigma * sigma))


def raw_derivs(k: np.ndarray | float, p: Iterable[float]) -> Tuple[np.ndarray, np.ndarray]:
    """First and second derivatives of raw SVI total variance with respect to k."""
    _, b, rho, m, sigma = [float(x) for x in p]
    k = np.asarray(k, dtype=float)
    x = k - m
    s = np.sqrt(x * x + sigma * sigma)
    w_prime = b * (rho + x / s)
    w_second = b * (sigma * sigma) / (s ** 3)
    return w_prime, w_second


def svi_density_g(k: np.ndarray | float, p: Iterable[float]) -> np.ndarray:
    """
    Gatheral-style risk-neutral density diagnostic for an implied total variance slice.

    A necessary practical butterfly-arbitrage check is g(k) >= 0 over the relevant domain.
    This grid check is a numerical diagnostic, not a symbolic proof over all real k.
    """
    k = np.asarray(k, dtype=float)
    w = raw_w(k, p)
    w_prime, w_second = raw_derivs(k, p)
    return (
        (1.0 - k * w_prime / (2.0 * w)) ** 2
        - (w_prime * w_prime / 4.0) * (1.0 / w + 0.25)
        + 0.5 * w_second
    )


def raw_to_jw(p: Iterable[float]) -> np.ndarray:
    """
    Convert raw SVI parameters to total-variance SVI-JW/SVI-JWK diagnostics.

    Returns [v, psi, p_put, c_call, vtilde].
    """
    a, b, rho, m, sigma = [float(x) for x in p]
    v = float(raw_w(np.array([0.0]), [a, b, rho, m, sigma])[0])
    sqrt_v = math.sqrt(v)
    S = math.sqrt(m * m + sigma * sigma)

    psi = b / (2.0 * sqrt_v) * (rho - m / S)
    p_put = b * (1.0 - rho) / sqrt_v
    c_call = b * (1.0 + rho) / sqrt_v
    vtilde = a + b * sigma * math.sqrt(max(0.0, 1.0 - rho * rho))

    return np.array([v, psi, p_put, c_call, vtilde], dtype=float)


def jw_to_raw(jw: Iterable[float]) -> np.ndarray:
    """
    Invert total-variance SVI-JW/SVI-JWK diagnostics to raw SVI parameters.

    This inverse is valid for the convention used in raw_to_jw().
    Returns [a, b, rho, m, sigma].
    """
    v, psi, p_put, c_call, vtilde = [float(x) for x in jw]

    if v <= 0.0 or p_put <= 0.0 or c_call <= 0.0:
        raise ValueError("Invalid SVI-JW parameters: v, p, and c must be positive.")

    sqrt_v = math.sqrt(v)
    b = sqrt_v * (p_put + c_call) / 2.0
    rho = (c_call - p_put) / (c_call + p_put)

    mu = rho - 4.0 * psi / (p_put + c_call)
    if abs(mu) >= 1.0:
        raise ValueError(f"Invalid SVI-JW parameters: |mu| >= 1. mu={mu}")

    denom = (
        1.0
        - rho * mu
        - math.sqrt(max(0.0, 1.0 - mu * mu)) * math.sqrt(max(0.0, 1.0 - rho * rho))
    )
    if denom <= 1e-14:
        raise ValueError(f"Invalid SVI-JW parameters: non-positive inverse denominator={denom}")

    S = (v - vtilde) / (b * denom)
    if S <= 0.0:
        raise ValueError(f"Invalid SVI-JW parameters: non-positive S={S}")

    m = mu * S
    sigma = S * math.sqrt(max(0.0, 1.0 - mu * mu))
    a = v - b * S * (1.0 - rho * mu)

    return np.array([a, b, rho, m, sigma], dtype=float)


def norm_cdf(x: np.ndarray | float) -> np.ndarray:
    """Vectorized standard-normal CDF using math.erf."""
    x = np.asarray(x, dtype=float)
    return 0.5 * (1.0 + np.vectorize(math.erf)(x / math.sqrt(2.0)))


def black_call_forward(K: np.ndarray, F: float, total_variance: np.ndarray) -> np.ndarray:
    """
    Undiscounted Black call price under a forward measure.

    The report uses F=100 and zero discounting to isolate the interpolation effect.
    Production validation should use production forwards, rates, dividends and strike conventions.
    """
    K = np.asarray(K, dtype=float)
    w = np.maximum(np.asarray(total_variance, dtype=float), 1e-16)
    sqrt_w = np.sqrt(w)
    k = np.log(K / F)
    d1 = (-k + 0.5 * w) / sqrt_w
    d2 = d1 - sqrt_w
    return F * norm_cdf(d1) - K * norm_cdf(d2)

def slice_summary(raw: np.ndarray) -> Dict[str, float]:
    """Return node-level slice diagnostics."""
    w = raw_w(K_GRID, raw)
    g = svi_density_g(K_GRID, raw)
    return {
        "min_total_variance": float(np.min(w)),
        "min_g_density_diagnostic": float(np.min(g)),
        "left_slope_b_1_minus_rho": float(raw[1] * (1.0 - raw[2])),
        "right_slope_b_1_plus_rho": float(raw[1] * (1.0 + raw[2])),
    }


def interpolate_jw(raw_left: np.ndarray, raw_right: np.ndarray, lam: float) -> np.ndarray:
    """Linearly interpolate SVI-JW parameters and convert back to raw SVI."""
    jw_left = raw_to_jw(raw_left)
    jw_right = raw_to_jw(raw_right)
    jw_mid = (1.0 - lam) * jw_left + lam * jw_right
    return jw_to_raw(jw_mid)


def analyze_dataset(df: pd.DataFrame, dataset_id: str) -> Dict[str, pd.DataFrame]:
    """Run node-level and off-node checks for one dataset."""
    sub = df[df["dataset_id"] == dataset_id].sort_values("maturity_years").reset_index(drop=True)

    node_rows = []
    for _, row in sub.iterrows():
        raw = row[["raw_a", "raw_b", "raw_rho", "raw_m", "raw_sigma"]].to_numpy(dtype=float)
        metrics = slice_summary(raw)
        node_rows.append({**row.to_dict(), **metrics})
    node_df = pd.DataFrame(node_rows)

    node_cal_rows = []
    offnode_rows = []

    for i in range(len(sub) - 1):
        left = sub.loc[i]
        right = sub.loc[i + 1]
        raw_left = left[["raw_a", "raw_b", "raw_rho", "raw_m", "raw_sigma"]].to_numpy(dtype=float)
        raw_right = right[["raw_a", "raw_b", "raw_rho", "raw_m", "raw_sigma"]].to_numpy(dtype=float)

        node_diff = raw_w(K_GRID, raw_right) - raw_w(K_GRID, raw_left)
        idx = int(np.argmin(node_diff))
        node_price_diff = black_call_forward(STRIKE_GRID, FWD, raw_w(K_GRID, raw_right)) - black_call_forward(
            STRIKE_GRID, FWD, raw_w(K_GRID, raw_left)
        )
        pidx = int(np.argmin(node_price_diff))
        node_cal_rows.append(
            {
                "left_label": left["maturity_label"],
                "right_label": right["maturity_label"],
                "left_T": left["maturity_years"],
                "right_T": right["maturity_years"],
                "min_node_total_variance_spread": float(node_diff[idx]),
                "k_at_min_node_tv_spread": float(K_GRID[idx]),
                "min_node_call_price_spread": float(node_price_diff[pidx]),
                "k_at_min_node_price_spread": float(K_GRID[pidx]),
            }
        )

        raw_path: List[np.ndarray] = [interpolate_jw(raw_left, raw_right, lam) for lam in LAMBDA_GRID]
        worst_tv = {"value": float("inf")}
        worst_price = {"value": float("inf")}
        worst_g = {"value": float("inf")}
        worst_left_vs_first = {"value": float("inf")}

        for j in range(len(LAMBDA_GRID) - 1):
            p_a = raw_path[j]
            p_b = raw_path[j + 1]
            w_a = raw_w(K_GRID, p_a)
            w_b = raw_w(K_GRID, p_b)
            tv_spread = w_b - w_a
            idx = int(np.argmin(tv_spread))
            if tv_spread[idx] < worst_tv["value"]:
                worst_tv = {
                    "value": float(tv_spread[idx]),
                    "k": float(K_GRID[idx]),
                    "lambda_left": float(LAMBDA_GRID[j]),
                    "lambda_right": float(LAMBDA_GRID[j + 1]),
                }

            price_spread = black_call_forward(STRIKE_GRID, FWD, w_b) - black_call_forward(STRIKE_GRID, FWD, w_a)
            pidx = int(np.argmin(price_spread))
            if price_spread[pidx] < worst_price["value"]:
                worst_price = {
                    "value": float(price_spread[pidx]),
                    "k": float(K_GRID[pidx]),
                    "lambda_left": float(LAMBDA_GRID[j]),
                    "lambda_right": float(LAMBDA_GRID[j + 1]),
                }

        for j, p in enumerate(raw_path):
            g = svi_density_g(K_GRID, p)
            idxg = int(np.argmin(g))
            if g[idxg] < worst_g["value"]:
                worst_g = {"value": float(g[idxg]), "k": float(K_GRID[idxg]), "lambda": float(LAMBDA_GRID[j])}
            # compare each off-node to first node of pair
            diff_first = raw_w(K_GRID, p) - raw_w(K_GRID, raw_left)
            idxf = int(np.argmin(diff_first))
            if diff_first[idxf] < worst_left_vs_first["value"]:
                worst_left_vs_first = {
                    "value": float(diff_first[idxf]),
                    "k": float(K_GRID[idxf]),
                    "lambda": float(LAMBDA_GRID[j]),
                }

        m_shift_ratio = abs(raw_right[3] - raw_left[3]) / min(raw_left[4], raw_right[4])
        offnode_rows.append(
            {
                "left_label": left["maturity_label"],
                "right_label": right["maturity_label"],
                "left_T": left["maturity_years"],
                "right_T": right["maturity_years"],
                "m_shift_ratio": float(m_shift_ratio),
                "worst_adjacent_lambda_tv_spread": worst_tv["value"],
                "k_at_worst_adjacent_lambda_tv_spread": worst_tv["k"],
                "lambda_left_worst_tv": worst_tv["lambda_left"],
                "lambda_right_worst_tv": worst_tv["lambda_right"],
                "worst_adjacent_lambda_call_price_spread": worst_price["value"],
                "k_at_worst_adjacent_lambda_price_spread": worst_price["k"],
                "lambda_left_worst_price": worst_price["lambda_left"],
                "lambda_right_worst_price": worst_price["lambda_right"],
                "worst_offnode_g": worst_g["value"],
                "k_at_worst_offnode_g": worst_g["k"],
                "lambda_at_worst_offnode_g": worst_g["lambda"],
                "worst_offnode_vs_leftnode_tv_spread": worst_left_vs_first["value"],
                "k_at_worst_offnode_vs_leftnode": worst_left_vs_first["k"],
                "lambda_at_worst_offnode_vs_leftnode": worst_left_vs_first["lambda"],
            }
        )

    return {
        "nodes": node_df,
        "node_calendar": pd.DataFrame(node_cal_rows),
        "offnode": pd.DataFrame(offnode_rows),
    }


def fig_to_base64(fig: plt.Figure) -> str:
    """Convert a matplotlib figure to base64 PNG."""
    buf = io.BytesIO()
    fig.savefig(buf, format="png", dpi=160, bbox_inches="tight")
    plt.close(fig)
    return base64.b64encode(buf.getvalue()).decode("ascii")


def make_pair_plot(df: pd.DataFrame, dataset_id: str, left_label: str, right_label: str) -> str:
    """Plot left node, selected off-node interpolated slices, and right node."""
    sub = df[df["dataset_id"] == dataset_id].sort_values("maturity_years").reset_index(drop=True)
    left = sub[sub["maturity_label"] == left_label].iloc[0]
    right = sub[sub["maturity_label"] == right_label].iloc[0]
    raw_left = left[["raw_a", "raw_b", "raw_rho", "raw_m", "raw_sigma"]].to_numpy(dtype=float)
    raw_right = right[["raw_a", "raw_b", "raw_rho", "raw_m", "raw_sigma"]].to_numpy(dtype=float)

    fig, ax = plt.subplots(figsize=(8.5, 4.6))
    ax.plot(K_GRID, raw_w(K_GRID, raw_left), label=f"{left_label} node", linewidth=2.4)
    for lam in [0.25, 0.50, 0.75]:
        p = interpolate_jw(raw_left, raw_right, lam)
        T = (1.0 - lam) * float(left["maturity_years"]) + lam * float(right["maturity_years"])
        ax.plot(K_GRID, raw_w(K_GRID, p), linestyle="--", label=f"SVI-JW interp λ={lam:.2f}, T={T:.3f}y")
    ax.plot(K_GRID, raw_w(K_GRID, raw_right), label=f"{right_label} node", linewidth=2.4)
    ax.set_title(f"{dataset_id}: node checks pass, off-node SVI-JW interpolation dips")
    ax.set_xlabel("log-moneyness k")
    ax.set_ylabel("total variance w(k)")
    ax.axvline(0.0, linewidth=0.8, linestyle=":")
    ax.legend(loc="best", fontsize=8)
    ax.grid(True, linewidth=0.4, alpha=0.5)
    return fig_to_base64(fig)


def df_to_html_table(df: pd.DataFrame, max_rows: int | None = None, float_fmt: str = "{:.8f}") -> str:
    """HTML table with rounded numeric values."""
    view = df.copy()
    if max_rows is not None:
        view = view.head(max_rows)
    for col in view.columns:
        if pd.api.types.is_float_dtype(view[col]):
            view[col] = view[col].map(lambda x: float_fmt.format(x))
    return view.to_html(index=False, escape=True, classes="data-table")
