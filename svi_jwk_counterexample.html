1 · Production Parameterisation (MV-111 §4.2.2)

The production parameter set is χK = {ν_t, ψ_t, κ_t, p_t, c_t}:

ν_t squared ATM volatility (ν_t ≥ 0) annualised, NOT total variance ψ_t ATM Skew (-p_t/2 ≤ ψ_t ≤ c_t/2) κ_t ATM Kurtosis (κ_t ≥ 0) ← the “K” in SVI-JWK p_t slope of Put Wing (p_t ≥ 0) c_t slope of Call Wing (c_t ≥ 0)

Forward mapping: raw SVI → production JWK (eqs 4.4–4.5, 4.12)

b = √(ν_t·T) · (c_t + p_t) / 2 ρ = (c_t − p_t) / (c_t + p_t) d = √(m² + σ²) ν_t = w(0) / T = [a + b(−ρm + d)] / T ψ_t = b(ρ − m/d) / (2√(ν_t·T)) ATM skew p_t = b(1 − ρ) / √(ν_t·T) c_t = b(1 + ρ) / √(ν_t·T) κ_t = [ w″(0)/2 − ψ_t² ] / (2√ν_t) where w″(0)=bσ²/d³ ← eq (4.12) post-Jun-2022

Inverse mapping: production JWK → raw SVI (eqs 4.3–4.11, current)

b = √(ν_t·T) · (p_t + c_t) / 2 eq (4.4) ρ = (c_t − p_t) / (c_t + p_t) eq (4.5) β = (c_t − p_t − 4ψ_t) / (c_t + p_t) = m/d eq (4.8) γ = b / [ 2 · (2√ν_t · κ_t + ψ_t²) ] eq (4.11): 2√ν_t·κ_t (annualised) m = β(1 − β²)γ eq (4.6) σ = (1 − β²)^(3/2) · γ from eq (4.7) d = √(m²+σ²) = (1−β²)·γ check a = ν_t·T − b(−ρm + d) eq (4.3)

Roundtrip error (raw→JWK→raw) < 10⁻¹⁵ at all five nodes. The γ denominator uses 2√ν_t·κ_t (annualised), not 2√(ν_t·T)·κ_t. Consistency: 4√ν_t·κ_t = w″(0) − 2ψ_t² holds exactly at each node.

Production interpolation (MV-111 §5.18.2)

Section 5.18.2 states: “In practice, the ATM volatility √ν_t is linearly interpolated rather than the total variance ν_t·T.” The production scheme:

For λ ∈ [0,1] between pillars T&sub1; and T&sub2;: √ν(λ) = (1−λ)·√ν&sub1; + λ·√ν&sub2; → ν(λ) = [√ν(λ)]² ψ(λ) = (1−λ)·ψ&sub1; + λ·ψ&sub2; κ(λ) = (1−λ)·κ&sub1; + λ·κ&sub2; p(λ) = (1−λ)·p&sub1; + λ·p&sub2; c(λ) = (1−λ)·c&sub1; + λ·c&sub2; T(λ) = (1−λ)·T&sub1; + λ·T&sub2; Each interpolated tuple is converted to raw SVI via the inverse mapping above.

MV-111 acknowledges this deviates from the calendar-safe standard (linear in total variance). The finding below demonstrates the consequence.

6 · Recommended Production Controls

For each adjacent pillar pair (T_left, T_right) at EOD: 1. Compute JWK params via raw_to_jwk() and check kappa_t ≥ 0, psi_t in [-p_t/2, c_t/2]. 2. Interpolate on a dense lambda-grid (≥ 200 steps): sqrt_nu(lam) = (1-lam)*sqrt(nu1) + lam*sqrt(nu2), nu(lam) = sqrt_nu(lam)^2 psi(lam), kappa(lam), p(lam), c(lam) all linear in lam 3. Convert via jwk_to_raw(): gamma = b / [2*(2*sqrt(nu)*kappa + psi^2)] 4. Test w(k, lam_j+1) - w(k, lam_j) ≥ -eps_TV for all k. [eps_TV = 1e-6] 5. Test fixed-strike call price monotonicity with actual F(T), D(T). 6. Escalation: flag any corridor where |delta_sqrt_nu| > 0.02 for enhanced review.

7 · Caveats

  • Synthetic dataset. Parameters are not from any specific market date or underlying.
  • The γ formula uses 2√ν_t·κ_t (current, post-Jun-2022). The pre-Jun-2022 formula used 2√(ν_t·T)·κ_t. Verify the EQ Server version currently deployed before relying on these results.
  • F=100, r=0. Production call price tests require actual F(T), D(T) for NK225, AAPL, Toyota 7203.T.
  • Interior butterfly violations (negative g interior) are not observed in this dataset.
