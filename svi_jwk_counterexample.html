1 · Raw SVI and JWK Convention

The raw SVI total-variance slice at log-moneyness k = log(K/F) is:

w(k) = a + b [ ρ(k − m) + √((k − m)² + σ²) ]

The SVI-JW diagnostics used in this report (total-variance convention, not implied-vol convention) are:

v = w(0) = a + b[−ρm + √(m² + σ²)] (ATM total variance) ψ = w'(0)/2 = b/(2√v) · [ρ − m/√(m² + σ²)] (ATM skew, scaled) p = b(1 − ρ)/√v (put-wing slope) c = b(1 + ρ)/√v (call-wing slope) ṽ = a + bσ√(1 − ρ²) (minimum total variance)

Interpolation in JWK space means: for λ ∈ [0,1], the interpolated vector is θ(λ) = (1−λ)·θ(T₁) + λ·θ(T₂) where θ = (v, ψ, p, c, ṽ). This vector is then converted back to raw SVI for each intermediate maturity.

Corrected inverse mapping (jw_to_raw) COR-1

The inverse mapping is derived analytically from the five equations above. Let:

b = √v · (p + c) / 2 ρ = (c − p) / (c + p) β = ρ − 2ψ√v / b ← equals m / √(m² + σ²) Then: σ = (v - ṽ) / ( b * [ (1-rho*beta)/sqrt(1-beta^2) - sqrt(1-rho^2) ] ) m = β · σ / √(1 − β²) a = ṽ − bσ√(1 − ρ²)

Roundtrip verification: applying raw_to_jw followed by jw_to_raw recovers all five raw parameters to machine precision (maximum absolute error < 10⁻¹⁵ across all five nodes).

2 · No-Calendar-Arbitrage Condition

For each fixed log-moneyness k, total variance must be non-decreasing in maturity:

w(k, T₂) − w(k, T₁) ≥ 0 for all k, T₂ > T₁

Equivalently, for fixed-strike European calls (at the same F, r — see isolation note below), call prices must be non-decreasing in maturity. Both conditions are tested.

Isolation assumption (all call price tests in this report): F = 100, r = 0, zero dividends. This isolates the interpolation effect from forward, discounting and dividend conventions. Production testing must use actual F(T), r(T) and dividend adjustments for NK225 / AAPL / Toyota conventions.

5 · Off-Node Interpolation Checks

All quoted nodes pass. The failures appear only on the interpolation path. Corridors 3M→6M and 6M→1Y produce calendar violations at off-node intermediate tenors. Corridor 1M→3M has m_shift_ratio = 4.62 and is an escalation-watch corridor.

Metrics explained COR-3 COR-4 COR-6

|Δm|/σ̄ (symmetric m-shift ratio)
|Δm|/σ̄ = |m_T2 − m_T1| / [(σ_T1 + σ_T2)/2]

Measures how far the SVI minimum shifts relative to the mean curvature width. High values indicate the interpolation path must traverse a large m-space relative to the curvature scale, creating non-monotone total variance paths. This is an escalation diagnostic; it is not a published no-arbitrage theorem. Symmetric denominator prevents order-dependence.

dw/dλ|₀ worst k (directional derivative at λ = 0⁺)
dw/dλ|₀(k) ≈ [w(k,λ=ε) − w(k,λ=0)] / ε, ε = 10⁻⁵

If negative at any k, the total variance surface begins decreasing immediately after departing the left node — the violation is analytically confirmed at the limit λ→0⁺, not merely a numerical grid artefact. The worst-k value is reported. If positive, the violation (if any) occurs in the corridor interior.

Adjacent-step ΔTV worst

Minimum of w(k,λ_j+1) − w(k,λ_j) over all (k, j). Negative values indicate a local calendar inversion between adjacent λ steps. Grid: 100 equal steps in [0,1].

ΔTV vs left-node worst

Minimum of w(k,λ) − w(k,T₁) over all (k, λ). Negative values indicate the interpolated surface drops below the quoted left-node total variance — a more severe model-implied calendar inconsistency against the quoted maturity. Under the simplified assumptions used here, this corresponds to a static calendar-arbitrage signal; production interpretation must use the actual forward, discounting and dividend conventions.

min g (interior λ only) COR-4

Butterfly density g(k) at interior interpolation steps only (λ ∈ (0,1), endpoints excluded). The prior version reported the minimum including endpoints, which always returned the right-node value — this was uninformative and has been corrected. If negative, butterfly arbitrage is introduced at an intermediate maturity not represented by any quoted pillar.

6 · Corridor Analysis

6M → 1Y: Most severe failure — violation confirmed at λ = 0⁺
  • dw/dλ|₀ = −0.023087 at k = −0.35: the total variance surface begins decreasing immediately after the 6M pillar. The violation is analytic, not a grid artefact.
  • worst ΔTV vs left-node = −0.003014 at k = −0.35, λ = 0.31: the interpolated slice at 31% of the way from 6M to 1Y has lower total variance than the quoted 6M slice at the same strike. Under the simplified assumptions F = 100, r = 0 and zero dividends, this represents a model-implied calendar-arbitrage signal against the 6M pillar.
  • worst ΔCall = −0.01348 at k = −0.17 (F=100, r=0): call price non-monotone in maturity at the same strike. In production with actual NK225 forwards, this region carries meaningful open interest.
  • |Δm|/σ̄ = 10.25: the SVI center-of-mass parameter m shifts 10× the mean curvature width across this corridor — the primary driver of the failure.
  • Interior butterfly: min g = 0.2976 (positive) — no butterfly arbitrage introduced at intermediate λ for this corridor.
3M → 6M: Weaker technical calendar-monotonicity breach in corridor interior (no onset at λ = 0⁺)

This corridor should be interpreted more cautiously than the 6M→1Y case. It shows a local total-variance monotonicity breach along the interpolation path, but it does not produce a material fixed-strike call-price inversion under the simplified test assumptions.

  • dw/dλ|₀ = +0.022481: positive — the surface is increasing at λ=0⁺. The violation occurs in the corridor interior, not at the left node.
  • worst adjacent-step ΔTV = −0.000128: a local inversion appears in the mid-corridor. TV spread vs the left node remains non-negative (worst = 0.000000), so the interpolated surface stays above the 3M node; the violation is between adjacent off-node λ steps only.
  • Call price spread: passes (within tolerance). The TV violation is small and does not translate to an economically significant call price inversion at this scale.
  • |Δm|/σ̄ = 0.80: relatively low. The failure here is driven by simultaneous changes in b and ρ rather than a large m-shift.
1M → 3M: Escalation watch (passes, but |Δm|/σ̄ = 4.62)

All metrics pass. However the m-shift ratio of 4.62 is moderately elevated. With more extreme market conditions (higher b, lower σ) the same corridor structure could generate violations. This corridor should be included in ongoing monitoring.

7 · Production Control Requirements

The dataset establishes that node-level absence of butterfly and calendar arbitrage is necessary but not sufficient to validate linear interpolation of SVI-JWK parameters. The following five-step control must be implemented for each adjacent node pair as part of the end-of-day surface acceptance test:

For each adjacent pillar pair (T_left, T_right): 1. Interpolate the SVI-JWK vector (v, ψ, p, c, ṽ) on a dense λ-grid (minimum 200 steps recommended; increase near-endpoints for dw/dλ|₀ accuracy). 2. Convert each interpolated JWK vector to raw SVI using the corrected jw_to_raw() mapping (as derived in Section 1). 3. Test w(k, λ_j+1) − w(k, λ_j) ≥ −ε_TV for all k and adjacent j. Recommended tolerance: ε_TV = 1e-6 (in total variance units). 4. Test fixed-strike Black price monotonicity under production forward F(T), discount factor D(T), dividend conventions. The F=100, r=0 test in this report isolates the interpolation effect; production tests must use actual market inputs. 5. Test g(k, λ) ≥ −ε_g at interior λ values only (endpoints already covered by the node-level butterfly check). Recommended ε_g = 1e-4. Escalation trigger: report |Δm|/σ̄ ≥ 5 to model risk for enhanced review, independent of pass/fail on tests 3–5.

8 · Caveats

  • This is a synthetic validation dataset. Parameter values are designed to demonstrate failures cleanly; they are not a reconstruction of any specific market date or underlying.
  • All call price tests use F = 100, r = 0, zero dividends. Production tests must incorporate actual forward curves, dividend schedules and discount factors for the relevant underlying (NK225 / AAPL / Toyota 7203.T).
  • The λ-grid uses 100 equal steps. For production, use ≥200 steps and supplement with local optimisation (gradient-based or golden-section search) around the worst grid point to ensure the reported minimum is not a grid artefact.
  • If the production SVI-JWK convention uses implied-vol (not total-variance) parameterisation for v, or a different definition of ψ, replace raw_to_jw() and jw_to_raw() with the production mapping before relying on these results.
  • The m_shift_ratio is an empirical escalation heuristic, not a published no-arbitrage bound. A corridor with low m_shift_ratio can still fail if b and ρ change significantly (as seen in 3M→6M).
  • Interior butterfly violations (negative g) are not found in this dataset. Their absence does not generalise: with more extreme parameter changes between pillars, butterfly arbitrage can be introduced at intermediate tenors independent of the calendar spread condition.
